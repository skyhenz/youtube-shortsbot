# PAKET KONTEN AUTO #031

## TOPIK
Efek Tetris: Kenapa Game Bisa Menghantui Pikiranmu

## SCRIPT NARASI
Pernah main game sampai malam, lalu pas mau tidur kamu masih melihat potongan Tetris jatuh di langit-langit? Ini disebut Efek Tetris. Otak sangat pandai memberi pola pada hal yang baru saja dipelajari. Pemain Tetris intens mulai melihat pola Tetris di mana-mana. Fenomena ini sebenarnya positif: artinya otakmu aktif memproses dan mengkonsolidasi informasi baru, bahkan saat kamu tidak sadar. Ini alasan kenapa belajar sebelum tidur sangat efektif untuk mengingat. Komentar jika kamu sudah tau ini!
